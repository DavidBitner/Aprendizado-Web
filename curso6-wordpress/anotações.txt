Passos para site em wordpress:
Configurações FTP
Configurações de Banco de dados == A lingua do banco deve ser latin2_general_ci. Quando criar um banco no localhost o usuário será root com a senha vazia.
Instalação do Wordpress na pasta do site == Download Wordpress > Extract Archives to website page
Instalação do tema == Appearence > Themes
Instalação de plugins == Pluggins > Installed Plugins
Arrumar configurações do site == Settings
Adicionar Páginas do site == Pages > Add New
Criar Menu == Apearence > Menus
Configurar Identidade do site == Appearence > Customize > Header > Site Identity
Configurar Cores == Appearence > Customize > Global > Colors
Configurar Tipografia == Appearence > Customize > Global > Typography
Configurar Menu Principal == Appearence > Customize > Header > Primary Menu
Configurar Cabeçalho Principal == Appearence > Customize > Header > Primary Header
Configurar cabeçalho Transparente == Appearence > Customize > Header > Transparent Header
Configurar Breadcrumb == Appearence > Customize > Breadcrumb
Configurar Rodapé == Appearence > Customize > Footer
Configurar Container == Appearence > Customize > Global > Container
Configurar Página inicial == Appearence > Customize > Homepage Settings
Editar e criar páginas == Pages > All Pages
Dividir o conteúdo da página por sessões
Fazer configurações da sessão == Layout, Spacing, Background, Border
Adicionar itens a sessão == Criar e configurar info box, formulário, carrossel etc...
Fazer as alterações necessárias no Microthemer
Criar rodapé
Criar Formulários == Criar os formulários do site através do plugin escolhido
Criar posts se necessário == Caso exista uma página de notícias ou eventos, os posts são a melhor forma de se organizar este tipo de conteúdo
Fazer a responsabilidade de todo o site == Mudar as configurações das páginas onde for necessário para tornar o site responsivo
Usar o plugin EWWW para redimensionar imagens == Ver se tem alguma imagem no site que está com tamanho desproporcional e consertar
Plugin: W3 total cache == Habilitar W3 total cache
Configuração avançada w3 total cache == Habilitar o page cache
Configuração avançada w3 total cache == Habilitar Minify
Configuração avançada w3 total cache == Habilitar Database Cache
Configuração avançada w3 total cache == Habilitar Object Cache
Configuração avançada w3 total cache == Habilitar Fragment Cache Method
Plugin: Duplicator == Fazer migração do site no computador para um servidor
Plugin: Companion Auto Update == Habilitar plugin para atualizações automáticas do site
Plugin: Cerber Security, Anti-spam & Malware Scan == Habilitar plugin de segurança e configurar o necessário
Plugin: WP Telegram (auto post and notifications) == Habilitar (se necessário) o plugin de interação com o Telegram e configurar o necessário
Plugin: Updraft Plus == Habilitar e configurar plugin de backup
Configuração avançada Updraft Plus == Excluir pasta de cache do backup
Plugin: Loco Translate == Habilitar (se necessário) plugin de multiidiomas no site

Ainda sobre Microthemer:
Microthemer tem o modo rascunho como opção, para não mudar a versão final do website antes de fazer todas as alterações desejadas.

Dentro das preferências do Microthemer existe a opção de importar CSS, para importar algo pronto como um botão flutuante.

Pelo Microthemer é possível fazer alterações finas de responsabilidade.
